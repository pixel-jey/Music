# 🎵 Android Music Player

A clean, lightweight, and practical open-source music player for Android.

## ✨ Features

* 🎧 **Local Playback**: Seamlessly scan and play local audio files.
* 💿 **Vinyl UI**: Classic retro vinyl record spinning animation while playing.
* 📈 **Optimized Build**: Pre-configured ProGuard rules for code shrinking and security.

## 📂 Project Structure

```text
app/data/android/Music/
├── libs/               # Third-party libraries and dependencies
├── res/                # UI layouts, drawables, and design assets
├── src/                # Core Java/Kotlin source code
├── proguard.cfg        # ProGuard obfuscation configuration
└── project.properties  # Android project properties
```

## 🚀 Quick Start

### Prerequisites
* Android APT
* JDK 17+

### Build Steps
1. Clone the repository:
   ```bash
   git clone github.com
   ```
2. Open the project in **Eclipse APT**.
3. Click **Run** to deploy the app onto your device or emulator.
